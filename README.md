# Text Area Project

A simple web project built with HTML, CSS and JavaScript.

## What it does

- User can type in a text area
- Shows a live character count
- Displays a warning when the 200 character limit is exceeded
- Prevents typing beyond the limit
- Submit button to submit the text

## Technologies Used

- HTML
- CSS
- JavaScript

## Files

| File | Purpose |
|---|---|
| `index.html` | Main HTML structure and styling |
| `proj2.js` | JavaScript logic for counter and alert |

## How to Run

1. Download or clone the project
2. Open `index.html` in your browser
3. Start typing in the text area
